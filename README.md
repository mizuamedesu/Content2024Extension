## 注意事項
1. APIリクエストベースの動作について
  - この拡張機能はAPIリクエストを監視する仕組みで動作しています
  - クライアントで一度も読み込まれていないツイートは検出できません
  - 過去のツイートを確認したい場合は、カウンターをクリックして表示される検索機能でクライアントに一度読み込む必要があります

2. 利用に関する免責事項
